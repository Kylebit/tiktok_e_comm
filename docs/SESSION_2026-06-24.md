## 4. MX 试点上品：`seller_sku=770003`（末四位 0003）

**商品：** 3 sheets/set 3D Arch Niche Wall Decal（PH `product_id=1732379849767749563`）

**使用的 TK 采集箱：** `collectBoxDetailId=2059296237`（原 MY 店已发布条目，认领到 MX）

### 4.1 踩坑与修复

| 失败原因（妙手 UI） | 根因 | 修复 |
|---------------------|------|------|
| 本地展示价最小 MXN1 | `priceIncludeVat` 为 null | 设 `price` = `priceIncludeVat` |
| 图片上传失败 kwcdn algo_check | 主图为 Temu CDN，TikTok 不支持 | 换 PH 在售 **ibyteimg.com** 主图 |
| 标题过长 | save 接口 ≤255 字符 | 截断 title |
| 包裹尺寸必填 | package L/W/H 为 null | 10×20×2 cm |

### 4.2 价格规则（已跑通）

```
MXN = PHP售价 × 1.2 × (PHP→CNY汇率) / (MXN→CNY汇率)
```

- PH 售价：**383 PHP**
- 加价 20% 后换算：**150.65 MXN**（销售价 + 本地展示价同值）
- 汇率：`config/settings.json` 中 PHP=0.118；MXN 按 0.36 CNY 参考

### 4.3 Open API 局限（重要）

- `save_move_collect_task` 仅返回 `{ result: success, data: [] }`
- **无发布失败原因、无任务 ID、无发布记录查询接口**（探测路径均为 `routeNotFound`）
- **最终成败必须在妙手 UI / TikTok 卖家中心确认**

**2026-06-24 结果：** 补图 + 补价后，**MX 店上架成功**（用户确认）。
